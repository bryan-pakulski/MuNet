# MuNet Overview

MuNet is a layered runtime for model development and deployment:

- **`munet_core`**: tensor/runtime/backend primitives.
- **`munet_training`**: training-focused APIs (`nn`, `optim`, losses).
- **`munet_inference`**: deployment-focused inference runtime.

## Choose your path

- **Python users**: start with [`API / Python`](../api/python.md) and [`Guides / Inference Engine`](../guides/inference_engine.md).
- **C++ users**: start with [`API / C++`](../api/cpp.md) and architecture pages.
- **Deployment users**: see serialization + inference workflow docs.
